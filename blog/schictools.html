Introducing scHiCTools!

scHiCTools is a computational toolbox for analyzing single-cell Hi-C data. It provides a comprehensive set of tools for processing, analyzing, and visualizing single-cell Hi-C data. The toolbox includes methods for quality control, normalization, dimensionality reduction, clustering, and visualization.

The toolbox is implemented in Python and R, and is compatible with both command-line and graphical user interfaces. It supports various data formats and can be easily integrated with existing analysis pipelines. The toolbox is available as open-source software and includes detailed documentation and tutorials.
